Diamond Color Chart

Referencia visual educativa para comprender la escala de color del diamante (D–Z). Incluye comparación lado a lado.

Cómo usar

  • Elegí una letra (D–Z) para ver el tono relativo.
  • Activá “comparación” para evaluar dos grados simultáneamente.
  • Interpretá considerando luz, fluorescencia y entorno.

Nota técnica

Este módulo utiliza un modelo visual simplificado. No reemplaza una evaluación gemológica profesional (condiciones controladas, instrumental y criterio de laboratorio).
