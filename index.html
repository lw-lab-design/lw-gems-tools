Diamond Color Chart

Referencia visual simplificada – uso educativo
Nota: el color percibido depende de la iluminación, fluorescencia y entorno. Este módulo utiliza un modelo visual simplificado.
D
Colorless
Apariencia extremadamente blanca. El tinte es imperceptible incluso bajo observación profesional. Utilizado en la mayoría de las piezas de joyería de alta gama.
E
Colorless
Apariencia muy blanca. El tinte es mínimo y difícil de detectar. Adecuado para la mayoría de las piezas de joyería.
F
Colorless
Blanco limpio con trazas apenas perceptibles. Mantiene una apariencia incolora en la mayoría de las aplicaciones de joyería.
